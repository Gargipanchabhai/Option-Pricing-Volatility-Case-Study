# Option-Pricing-Volatility-Case-Study
Dynamic Volatility & Option Pricing Case Study

This project extends Bao (2024) by incorporating rolling volatility into a binomial option pricing framework.

Overview

Traditional models assume constant volatility.
This study evaluates how dynamic volatility affects option pricing and hedging decisions.

Key Features
	•	Rolling volatility analysis
	•	Binomial tree option pricing
	•	Mispricing comparison
	•	Payoff distribution analysis
	•	Strategic hedging framework

Data Source

Yahoo Finance — Apple Inc. (AAPL)

	•	NumPy & Pandas
	•	Matplotlib
	•	Jupyter Notebook
